# CA Music Generator - 一维细胞自动机音乐生成器

一个基于Godot 4.x的可视化音乐生成器，使用一维细胞自动机（如Rule 30）的演化状态来生成音乐。

## 功能特性

- **一维细胞自动机**：支持256种基本规则（0-255）
  - Rule 30 - 混沌模式
  - Rule 90 - 谢尔宾斯基三角形
  - Rule 110 - 图灵完备
  - Rule 184 - 交通流模型
  - 以及所有其他规则

- **多规则混合模式**：
  - **单一规则** - 使用单个规则演化
  - **混合模式** - 两个规则随机混合应用
  - **交替模式** - 两代之间交替使用两个规则
  - **位置模式** - 左右两半使用不同规则

- **交互式扰动**：
  - 点击或拖拽网格来翻转细胞状态
  - 实时影响自动机演化和音乐

- **音乐生成**：
  - 多种音阶可选（五声音阶、大调音阶、布鲁斯音阶等）
  - 实时音频合成（正弦波、方波、三角波）
  - 细胞位置映射到音高
  - 音频优化防止爆音

- **交互控制**：
  - 规则滑块（0-255）
  - 第二规则滑块（混合模式）
  - 速度控制
  - 音阶选择
  - 规则模式选择
  - 暂停/继续
  - 静音
  - 重置
  - 随机规则

## 快捷键：
  - **空格键**：暂停/继续
  - **R键**：重置
  - **鼠标点击/拖拽**：翻转细胞

## 运行方式

2. 打开项目文件夹
3. 点击运行按钮（或按F5）

## 技术实现

### 细胞自动机原理

每个细胞的下一个状态由其左右邻居和自身的当前状态决定，形成8种可能的模式（000到111）。每种模式对应规则二进制表示中的一位，决定下一代细胞是存活（1）或死亡（0）。

### 音乐映射

细胞自动机中的活跃细胞（值为1）被映射到音阶中的音符：
- 细胞位置 → 音符在音阶中的索引
- 从左到右对应从低到高的音高
- 支持多八度音程
